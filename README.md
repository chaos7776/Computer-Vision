### 全连接卷积网络

2014年，加州大学伯克利分校的Long等人在[Fully Convolutional Networks for Semantic Segmentation](https://arxiv.org/pdf/1411.4038.pdf)提出的完全卷积网络，推广了原有的CNN结构，在不带有全连接层的情况下能进行密集预测。
